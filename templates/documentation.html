📜 NHRC SOP Portal Documentation

Overview

This platform helps manage Standard Operating Procedures (SOPs), amendments, approvals, user management, and system settings securely and efficiently.

Modules

  • 📂 Documents: View, filter, and download SOPs.
  • 📤 Upload SOP: Upload new SOPs into the system (admin or HOD roles).
  • 📊 Dashboard: View live statistics and document status.
  • 🛡️ Admin Panel: Manage users, API keys, settings, and system logs.
  • ✏️ Amendments: Raise and track amendment requests to existing SOPs.

Access Roles

Permissions are based on user roles: Admin, HOD, Staff. Each has specific actions they can perform.
